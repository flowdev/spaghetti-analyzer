spaghetti-analyzer -	Package spaghetti-analyzer is the main and root package of the project.
├── analdata -	Package analdata contains the central data types of the spaghetti-analyzer.
├── deps -	Package deps analyzes and reports the dependencies between Go packages.
├── doc -	Package doc documents of the Go package structure as a dependency table.
├── parse -	Package parse does the actual parsing of Go code.
├── stat -	Package stat creates statistics for analyzing the Go package structure.
├── tree -	Package tree creates a directory tree with information about Go packages.
└── x -	Package x contains tool packages.
    ├── dirs -	Package dirs is a tool for directory handling.
    └── pkgs -	Package pkgs is a tools for simpler handling of parsed Go packages.
